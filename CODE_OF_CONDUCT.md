# Code of Conduct

This project adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

We expect everyone participating in this project — through issues, pull requests, comments, or any other form of contribution — to engage respectfully and constructively. Harassment, discrimination, and abusive behavior are not tolerated.

## Enforcement

Report concerns to the maintainer at `lucas.diassantana@gmail.com`. All reports are reviewed confidentially. The maintainer may take any action deemed appropriate, up to and including removing contributions and blocking future participation.

For the full set of expectations, enforcement guidelines, and attribution, see the canonical Contributor Covenant 2.1 text at the link above.
